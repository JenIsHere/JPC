Introduction

Project Statement

According to "A Place to Grow-Growth Plan", the municipalities of the Greater Golden Horseshoe (GGH) region are required to update their development plans through 2051 by the end of June 2022.

If allowed, the inreasingly imminent urban sprawl will enter the Contested Countryside lands, which would result in the loss of natural heritage and agricultural lands.

Requirements

Visualizing the Contested Countryside to shed a light on how little land is left.

Performing analyses and producing results to depict the potential impact of urban sprawl into these lands.

Creating various mediums to share the results in order to raise awareness regarding this matter.

Deliverables

A Large Format Map

A Web Map with analyses results

A Slide Deck displaying the layers

About This Project

This project is created in order to develop the necessary skillsets in Web GIS Development as well as creating an appealing webpage. Team JPC will implement the acquired knowledge of open-source software such as Leaflet and Mapbox to create a solution. This knowledge will be put to further use for Collaborative Project during the Spring 2022 term.
